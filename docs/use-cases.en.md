# Use Cases and Application Matrix

## Guiding Idea

The rule and profile system does not improve answer quality in a universal way.

Its main value is improving the **fit between answer quality and task type**.

## Quick Matrix

| Question Type | Recommended Combination | Why |
|---|---|---|
| Simple factual question | Comm Stop or Standard + Strict | direct, understandable, low overhead |
| Precise technical explanation | Standard + Strict | strong clarity/discipline balance |
| Difficult conceptual question | Expert + SCI A + Strict | cleaner structure, stronger methodological control |
| Fundamental question | Expert + SCI B + Strict | deeper perspective work |
| Testing a controversial claim | Sparring + Strict | stronger opposition and weakness detection |
| Testing hype/breakthrough claims | Sparring + SCI G + Strict or Expert + SCI E + Strict | skepticism, failure modes, uncertainty work |
| Brainstorming / ideation | Sandbox + Explore | widest search space |
| Didactic simplification | Standard + Strict or Briefing + Strict | clearer, reduced presentation |
| Decision preparation | Briefing + Strict | compact orientation |

## Default Recommendations

- Explain: `Profile Standard + Strict on`
- Check: `Profile Expert + SCI on` (variant A) + `Strict on`
- Challenge: `Profile Sparring + Strict on`
- Explore: `Profile Sandbox + Explore on`
- Condense: `Profile Briefing + Strict on`
- Normal mode: `Comm Stop`

## Five Example Questions

1. What is time? -> Expert + SCI A/B + Strict
2. Are homework assignments pedagogically outdated? -> Sparring + Strict or Expert + SCI E + Strict
3. Sketch three unconventional physics lessons on energy -> Sandbox + Explore
4. Explain speed vs acceleration for grade 8 -> Standard + Strict or Briefing + Strict
5. Evaluate claim "Model X is a breakthrough" -> Sparring + SCI G + Strict

## Practical Rules

1. For most cases, Standard + Strict is enough.
2. Expert is worth it when precision matters more than reading flow.
3. Use Sparring only when genuine opposition is desired.
4. Sandbox is for search space, not finality.
5. SCI B is powerful but often too expensive.
6. SCI recurse is a specialist tool, not a default.
7. Color changes presentation, not epistemic quality.
