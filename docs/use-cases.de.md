# Use Cases und Anwendungsmatrix

## Leitidee

Das Regel- und Profilsystem verbessert nicht pauschal die Antwortqualitaet.

Es verbessert vor allem die **Passung der Antwortqualitaet zur Aufgabe**.

## Schnellmatrix

| Fragetyp | Empfohlene Kombination | Warum |
|---|---|---|
| Einfache Sachfrage | Comm Stop oder Standard + Strict | direkt, verstaendlich, wenig Overhead |
| Fachlich praezise Erklaerung | Standard + Strict | gute Balance aus Klarheit und Disziplin |
| Schwierige Begriffsfrage | Expert + SCI A + Strict | saubere Struktur, hohe methodische Belastbarkeit |
| Grundsatzfrage | Expert + SCI B + Strict | tiefe Perspektivenpruefung |
| Kontroverse These pruefen | Sparring + Strict | staerkere Gegenrede und Schwachstellenpruefung |
| Hype-/Durchbruch-Behauptung pruefen | Sparring + SCI G + Strict oder Expert + SCI E + Strict | Skepsis, Fehlermodi, Unsicherheitsarbeit |
| Brainstorming / neue Ideen | Sandbox + Explore | groesster Suchraum |
| Didaktische Vereinfachung | Standard + Strict oder Briefing + Strict | klare, reduzierte Darstellung |
| Entscheidungsvorbereitung | Briefing + Strict | kompakte Orientierung |

## Standardempfehlungen

- Erklaeren: `Profile Standard + Strict on`
- Pruefen: `Profile Expert + SCI on` (Variante A) + `Strict on`
- Gegenpruefen: `Profile Sparring + Strict on`
- Erkunden: `Profile Sandbox + Explore on`
- Verdichten: `Profile Briefing + Strict on`
- Normalmodus: `Comm Stop`

## Fuenf Musterfragen

1. Was ist Zeit? -> Expert + SCI A/B + Strict
2. Sind Hausaufgaben paedagogisch ueberholt? -> Sparring + Strict oder Expert + SCI E + Strict
3. Drei unkonventionelle Physikstunden zu Energie -> Sandbox + Explore
4. Geschwindigkeit vs. Beschleunigung fuer Klasse 8 -> Standard + Strict oder Briefing + Strict
5. Modell X ist ein Durchbruch -> Sparring + SCI G + Strict

## Wichtige Praxisregeln

1. Fuer die meisten Faelle reicht Standard + Strict.
2. Expert lohnt sich, wenn Praezision wichtiger ist als Lesefluss.
3. Sparring nur waehlen, wenn echte Gegenpruefung gewuenscht ist.
4. Sandbox ist fuer Suchraum, nicht fuer Abschlusskraft.
5. SCI B ist maechig, aber oft zu teuer.
6. SCI recurse ist Spezialwerkzeug, nicht Standard.
7. Color betrifft Darstellung, nicht Erkenntnisqualitaet.
